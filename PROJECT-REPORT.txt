  3. نکات فنی کلیدی
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  [LANG]   pa_current_lang() فقط از cookie pa_lang استفاده می‌کند
           Polylang این cookie را override نمی‌کند
           Cookie یک سال اعتبار دارد

  [QUERY]  WP_Query → همیشه new WP_Query($args_array) با array جداگانه
           همیشه instanceof WP_Query قبل از have_posts() چک شود
           هرگز از orderby=rand استفاده نشود (integer برمی‌گرداند!)

  [SINGLE] صفحات single از get_queried_object_id() استفاده می‌کنند
           بدون while(have_posts()) loop اصلی
           Secondary queries تداخل ایجاد نمی‌کنند

  [PLAYER] Audio player همیشه direction:ltr دارد
           صرف‌نظر از زبان سایت (FA یا EN)

  [CSS]    fixes.css همیشه آخرین CSS است
           همه override ها باید اینجا باشند

  [GIT]    فایل‌های موقت (setup-*.php، fix-*.php) را commit نکن
           همیشه قبل از push: فایل‌های temp را خالی کن
